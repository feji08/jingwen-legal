## 一、我们收集和使用的信息

### 1.1 本地存储的信息

以下信息**仅存储在您的本地设备上**，不会上传至我们的服务器：

| 信息类型    | 用途        | 存储方式                      |
| ------- | --------- | ------------------------- |
| 姓名      | 填写版权投诉表单  | 操作系统安全存储加密（Windows DPAPI） |
| 身份证号    | 填写版权投诉表单  | 操作系统安全存储加密                |
| 电子邮箱    | 填写版权投诉表单  | 操作系统安全存储加密                |
| 手机号码    | 填写版权投诉表单  | 操作系统安全存储加密                |
| 书名及笔名   | 检索和管理版权投诉 | 本地 SQLite 数据库             |
| 授权材料文件  | 版权投诉附件    | 本地 SQLite 数据库             |

### 1.2 传输至服务器的信息

本软件仅在以下场景与我们的服务器通信：

| 场景       | 传输内容               | 说明                                                          |
| -------- | ------------------ | ----------------------------------------------------------- |
| 激活许可证    | 许可证密钥、设备指纹哈希值      | 设备指纹由主机名、操作系统类型和网卡 MAC 地址经 SHA256 哈希后截取前 32 位生成，无法还原为原始硬件信息 |
| 获取投诉规则更新 | JWT 令牌（仅含密钥前缀和有效期） | 不含任何个人身份信息                                                  |

### 1.3 我们不收集的信息

- 我们**没有用户账号体系**，不收集您的用户名或密码
- 我们**不使用任何第三方分析、统计或崩溃追踪服务**
- 我们**不使用 Cookie 或类似追踪技术**
- 我们**不会读取您的浏览器历史记录**
- 您在搜索引擎上的检索记录**不会上传至我们的服务器**

## 二、信息的存储与安全

### 2.1 存储位置

您的个人敏感信息（姓名、身份证号、邮箱、手机号）使用操作系统级加密（Windows DPAPI / macOS Keychain）存储在本地设备的 SQLite 数据库中。我们的服务器不存储上述任何信息。

### 2.2 传输安全

本软件与服务器之间的所有通信均通过 HTTPS 加密传输。

### 2.3 本地 HTTP 服务

本软件在本地运行一个仅限本机访问（127.0.0.1）的 HTTP 服务，用于与浏览器插件通信。该服务使用随机令牌验证，每次启动应用时自动更换，不接受外部网络连接。

## 三、信息的使用目的

我们收集的信息仅用于以下目的：

1. **许可证验证**：通过设备指纹和许可证密钥验证您的软件使用权
2. **规则更新**：从服务器获取最新的平台投诉规则
3. **自动填写投诉表单**：使用您本地存储的个人信息和书籍信息协助填写各平台投诉表

## 四、信息的共享与披露

- 我们**不会**将您的信息出售、出租或以其他方式提供给任何第三方
- 当您向各搜索引擎或平台提交版权投诉时，您的个人信息由您的浏览器直接提交至目标平台，**不经过我们的服务器中转**
